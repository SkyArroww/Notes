## 1.3 CS50P Introduction to Programming with Python
* 35h
*  Python 的基础与进阶语法，以及各种 “Pythonic” 的编程方法。此外对于 Python 特色的代码库、代码测试以及错误处理，该课程也会较为深入地探究
## 1.4 CS61A: Structure and Interpretation of Computer Programs
* 50h
* 抽象将是这门课的一大主题，你将学习到函数式编程、数据抽象、面向对象等等知识来让你的代码更易读，更模块化。
* 学习编程语言也是这门课的一大内容，你将会掌握 Python、Scheme 和 SQL 这三种编程语言，在它们的学习和比较中，相信你会拥有快速掌握一门新的编程语言的能力

## 1.5 CS106B/X: Programming Abstractions in C++
* 60h
* 通过 C++ 语言让学生在实际的编程作业里培养通过编程抽象解决实际问题的能力，同时也会涉及一些简单的数据结构和算法的知识
## 1.6 CS106L: Standard C++ Programming
* 20h
* 深入到很多标准 C++ 的特性和语法，让你编写出高质量的 C++ 代码。例如 auto binding, uniform initialization, lambda function, move semantics，RAII 等技巧
## 1.7 CS110L: Safety in Systems Programming
* 30h 
* 系统地掌握多进程、多线程、基于事件驱动的并发等若干并发技术，并在第二个 Project 中比较它们各自的优劣
## 1.8 Amirkabir University of Technology 14002: Advanced Programming Course
* 50h
* C++课程。课程的 homework 质量很高，每个 homework 相互独立结构简单，且有完善的单元测试，非常适合用来学习 C++ 的编程
## 1.7 CS431: Concurrent Programming
* 50h
* 理论部分聚焦于建立并发情形下的编程模型，而实践部分则主要是理解 Rust 相关库中锁与无锁数据结构的实现原理
* 这门课程同时配有一系列代码量不大但并不简单的作业来巩固你对并发编程的理解，从基于锁的并发安全缓存设计和链表，到无锁的哈希表和著名的 [hazard pointer](https://ieeexplore.ieee.org/document/1291819). 和众多高质量课程一样，这些作业质量较高且配有详细的本地测试

* 100h

## 4.3 Stanford CS 346: Database SystemImplementation
* 150h 
* The recordmanagement component:记录管理组件。The indexcomponent:B+索引管理。The System ManagementComponent: ddl语句、命令行工具、数据加载命令、元数据管理。The QueryLanguage Component:在这个部分需要实现 RQL Redbase 查询语言。RQL要实现select、insert、delete、update 语句。Extension Component:除了上述数据库系统的基本功能组件，还需要实现一个扩展组件，可以是Blob类型、网络模块、连接算法、CBO 优化器、OLAP.事务等
